# Text-to-SQL AI Agent
**Tech Stack**
- Crew AI
- Langchain SQL Database Toolkit
- Llama 3.3 70B Instruct Turbo LLM (Together AI API)

**Installation**
- Install Crew AI and dependencies
- Install Langchain and dependencies

**Streamlit App**
```bash
streamlit run sqlcodeagent.py
```
